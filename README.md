**Options:**
  **\-i**, **\--input** <input>    Input FASTA or FASTQ file, possibly gzipped
  **\-o**, **\--output** <output>  Output FASTA or FASTQ file, possibly gzipped
  **\-k** <k>                 k-mer length of the unitigs
  **\-h**, **\--help**             Print help
